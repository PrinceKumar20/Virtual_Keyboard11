Now a days computer vision has developed to an instant where it can recognize image processing algorithms, people are using this in many aspects in their life by identifying and processing objects in images and videos for face recognition, color detection etc. Main purpose of making such applications is Automating the key click functionality eventually discarding the electronic equipment. Hence, control typing on a virtual keyboard by using various fingertip movements. To press any key on Gesture Keyboard just bring (8-index), (12-mid) finger positions close. mediapipe module which draws pipeline to hand gestures to perform a click,both hands can be operated.
